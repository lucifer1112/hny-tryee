# শুভ নববর্ষ (Shubho Noboborsho) - Greeting Webpage 🌺

A personalized, beautiful mini-website created to wish a Happy Bengali New Year. 

## ✨ About The Project

This repository contains a simple, elegant webpage designed with a warm aesthetic, floating sparklers, and beautiful Bengali typography. It was built specifically to share a sweet, heartfelt New Year message.

**Message included:**
> “শুভ নববর্ষ, আমার প্রিয় Trayee ❤️
> নতুন বছর তোমার জীবনে নিয়ে আসুক অনেক সুখ, ভালোবাসা আর সুন্দর মুহূর্ত।
> তুমি আমার জীবনের খুব স্পেশাল একজন, আর তোমার হাসিটা সবসময় এমনই থাকুক।
> আমাদের বন্ধুত্বটা যেন এই নতুন বছরেও আরও গভীর আর সুন্দর হয়।
> সবসময় পাশে থেকো—কারণ তোমার সাথে থাকলে সবকিছুই একটু বেশি ভালো লাগে। 💫”

You can view the live website here:

## 🛠️ Built With

* **HTML5**: For the structure of the greeting card.
* **CSS3**: For the warm gradient background, glassmorphism card effect, and responsive design.
* **JavaScript**: For the gentle floating sparkle animation in the background.
* **Google Fonts**: Uses the `Hind Siliguri` font for elegant Bengali typography.

## 📂 File Structure

* `index.html` - The main structure and text of the webpage.
* `style.css` - The styling, layout, and CSS animations.
* `script.js` - The logic for generating the floating sparkles.
* `README.md` - Project documentation.

## 💡 How to Use

If you want to run this locally, simply download the files and open `index.html` in any modern web browser. No server setup is required!
